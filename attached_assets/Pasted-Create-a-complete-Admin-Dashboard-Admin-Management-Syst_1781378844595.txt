Create a complete **Admin Dashboard & Admin Management System** for the platform.

## Objective

Build a powerful admin panel where the platform owner can monitor, manage, analyze, and control every part of the application.

The admin should have complete visibility and control over users, creators, brands, content, analytics, verification, SEO, and platform activity.

---

# Admin Authentication

Create a secure admin login system.

Features:

* phone number/ password
* Role-based access control
* Secure admin routes
* Session management
* Logout option

Admin roles:

* Super Admin (full access)
* Admin (limited access)
